# App-fazendinha
O projeto é um aplicativo web de gestão rural chamado APP-Fazendinha desenvolvido para funcionar diretamente no navegador, inclusive offline. A plataforma centraliza o controle operacional da fazenda em uma interface simples, rápida e visual, permitindo acompanhar animais, pastos, manejos, movimentações, finanças e insumos em um único sistema.
