# App-fazendinha
O projeto é um aplicativo web de gestão rural chamado APP-Fazendinha desenvolvido para funcionar diretamente no navegador, inclusive offline.
A plataforma centraliza o controle operacional da fazenda em uma interface simples, rápida e visual, permitindo acompanhar animais, pastos, manejos, movimentações, finanças e insumos em um único sistema acessível de qualquer dispositivo com internet.
Principais funcionalidades:


gerenciamento completo de pastos


controle de lotação e estado das áreas


medição de áreas por satélite


cadastro individual de animais


registro de nascimentos


movimentação entre pastos


histórico de manejos sanitários


controle de sal mineral e consumo


painel financeiro


alertas automáticos da fazenda


previsão do tempo integrada


funcionamento offline


armazenamento local de dados


interface otimizada para celular


O sistema foi desenvolvido utilizando:


HTML


React


JavaScript


Leaflet para mapas


O objetivo do APP-Fazendinha é oferecer uma ferramenta moderna, leve e prática para gerenciamento agropecuário, simplificando a rotina da fazenda e permitindo controle operacional em tempo real de forma acessível e intuitiva.
